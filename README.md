# create-react-project

프로젝트 세팅 전에 변경할 것!

- `package.json` 에 `name`, `author`
- `codecept.conf.ts` 의 `name`

## Install dependencies

```bash
npm install
```

## Run web server

```bash
npm start
```

## Run tests

```bash
npm test
```

## Lint and fix all

```bash
npm run lint
```

## Build

```bash
npm run build
```
